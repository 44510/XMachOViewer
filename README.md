# MachO file viewer/editor for Windows, Linux and macOS.

### Features

- Heuristic scan
- String viewer
- Hex viewer
- Disasm viewer(x86/64,ARM,PPC,m68k)
- Entropy viewer
- Hash viewer
- Crypto search
- Name demangling
